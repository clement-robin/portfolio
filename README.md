# Portfolio

My portfolio present : myself, my experiences, skills and projects. 

## Techs used

- React
- Tailwind CSS
- Three.js


## Installation

To run in local

```bash
npm run dev
```

To run build dependencies

```bash
npm run build
```

To deploy on Github Pages

```bash
npm run deploy
```
