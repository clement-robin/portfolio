# Portfolio

My portfolio present me, my experiences, my skils and projects.

## Techs used

- React
- Tailwind CSS
- Three.js


## Instalation

To run in local

```bash
  npm run test
```

To run build dependencies

```bash
  npm run build
```

To deploy on Github Pages

```bash
  npm run deploy
```
